Numerical quadrature

Trapezoidal rule: \begin{equation*} \int_{x_{0}}^{x_{n}} f(x) dx \approx \frac{h}{2} \left( f(x_{0}) + f(x_{n}) \right) + h \sum_{k=1}^{n-1} f(x_{k}) \end{equation*}

Trapezoidal rule with end corrections using the first derivative: \begin{equation*} \int_{x_{0}}^{x_{n}} f(x) dx \approx \frac{h}{2} \left( f(x_{0}) + f(x_{n}) \right) + h \sum_{k=1}^{n-1} f(x_{k}) - \frac{h^{2}}{12} \left( f^{\prime}(x_{n}) - f^{\prime}(x_{0}) \right) \end{equation*}

Trapezoidal rule with end corrections using the first and third derivative: \begin{equation*} \int_{x_{0}}^{x_{n}} f(x) dx \approx \frac{h}{2} \left( f(x_{0}) + f(x_{n}) \right) + h \sum_{k=1}^{n-1} f(x_{k}) - \frac{h^{2}}{12} \left( f^{\prime}(x_{n}) - f^{\prime}(x_{0}) \right) + \frac{h^{4}}{720} \left( f^{\prime\prime\prime}(x_{n}) - f^{\prime\prime\prime}(x_{0}) \right) \end{equation*}

Gauss-Legendre quadrature: \begin{equation*} \int_{-1}^{1} f(x) dx \approx \sum_{k=1}^{n} w_{k} f(x_{k}) \end{equation*}
