Key combinations allow you to do things easier and faster with the GNU shell, Bash, which is the default on almost any Linux system. Below is a list of the most commonly used features: you are strongly suggested to make a habit out of using them, so as to get the most out of your Linux experience from the very beginning.

Ctrl + A Move cursor to the beginning of the command line.
Ctrl + B Move cursor backward one character on the current line.
Ctrl + C End a running program and return the prompt.
Ctrl + D Log out of the current shell session, equal to typing exit or logout.
Ctrl + E Move cursor to the end of the command line.
Ctrl + F Move cursor forward one character on the current line.
Ctrl + G Leave history searching mode without running a command.
Ctrl + H Move cursor to the end of the command line.
Ctrl + J Lets you copy current matched command to command line without running it.
Ctrl + K Cut the line from the current position to the end of the line, adding it to the clipboard.
Ctrl + L Clear this terminal.
Ctrl + M
Ctrl + N Shows next executed command (similar to down arrow).
Ctrl + O
Ctrl + P
Ctrl + R Search command history.
Ctrl + U
Ctrl + W
Ctrl + Z Suspends (pause) currently running foreground process.
