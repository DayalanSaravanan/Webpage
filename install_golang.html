Golang Installation

Instructions to download and install the latest Golang version in Debian 10

  1. Download the Go installer:

    $ wget -c https://dl.google.com/go/go1.17.7.linux-amd64.tar.gz

  2. Extract the archive you downloaded into /usr/local, creating a Go tree in /usr/local/go.

    3. Important: This step will remove a previous installation at /usr/local/go, if any, prior to extracting. Please back up any data before proceeding.

    For example, run the following as root or through sudo:

    $ rm -rf /usr/local/go && tar -C /usr/local -xzf go1.17.7.linux-amd64.tar.gz

  3. Add /usr/local/go/bin to the PATH environment variable:

    You can do this by adding the following line to your $HOME/.profile or /etc/profile (for a system-wide installation):

    export PATH=$PATH:/usr/local/go/bin

    Note:Changes made to a profile file may not apply until the next time you log into your computer. To apply the changes immediately, just run the shell commands directly or execute them from the profile using a command such as source $HOME/.profile

  4. Verify that you've installed Go by opening a command prompt and typing the following command:

    $ go version

  5. Confirm that the command prints the installed version of Go.

    go version go1.17.7 linux/amd64
